Analytics as Service on Hadoop

Application to provide analytics over any type of data. The application is developed over Hadoop Big Data framework which provides parallel processing using low end commodotity servers. The sample applications are deployed amazon EC2 cluster with 3.0GHZ quad core with 4 GB of RAM each. The sample application is capable of processing 10TB of twitter and analytics data.

As testing tool

This application provides a nice way to monitor the testing status of the web application

Sample Business Cases Deployed

Small Business Website/Application Monitor

The application is capable of monitoring website using various data sources. The application is developed to collect the click stream data using script included in the webpages and log data from the servers

Features

Reports Available for Monitor

  • Unique Visits Per day/Month/3Month
  • Visitor User Geography(Using ArcGis Maps)
  • Percentile of Page Access
  • Click information on individual page
  • User browser data
  • User Operating System

    • Server log monitor
